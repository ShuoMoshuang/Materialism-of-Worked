# Materialism-of-Work作用唯物主义：实践唯物主义的继承与发展
> 本仓库为**作用唯物主义**原创哲学体系的开源迭代仓库，旨在以一套贯通自然演化、个体认知与社会历史的一元哲学框架，回应当下时代的现实困境与理论缺口，欢迎所有对马克思主义哲学、系统科学、认知科学与社会现实问题感兴趣的参与者共同讨论、修正、完善与拓展。

---

## 一、时代困境：唯物主义的失语与普通人的双重困境
我们正处在一个充满矛盾的时代：物质生产力达到了人类历史的顶峰，全球化与技术进步让物质财富的生产效率空前提升，但普通人的生存困境并未随之消解，反而陷入了更深的双重内耗之中。

一方面，是**物质层面的结构性困境**：跨国资本主义通过产业跨国转移，将阶级矛盾与剥削风险向全球边缘国家转嫁，资产阶级通过内部圈层化完成了统治秩序的自我巩固，而绝大多数普通人陷入了“越努力越内卷”的循环——劳动回报与劳动付出持续脱钩，资源分配呈现极端的过饱和集中，少数人掌握的资源远超其可支配的边界，而多数人的资源仅能维持基本生存，远不足以支撑其自由发展的需求。传统的辩证唯物主义理论，在被教条化、机械化的解读后，面对这种跨地域、多层级、高度系统化的新型剥削结构，出现了明显的解释力缺口，无法为普通人提供清晰的认知工具与行动指引。

另一方面，是**意义层面的虚无化困境**：元叙事的崩解带来了普遍的价值真空，后现代的碎片化语境让个体失去了稳定的意义锚点。普通人既被剥削阶级的意识形态裹挟，陷入“精致生活”“消费主义”的梦游式生存，把依附于剥削结构的价值标准当成了自身的生存目标；又被各类唯心主义、玄学、虚无主义的叙事包围，在“躺平”与“内耗”的两极之间摇摆，对现实的无力感转化为对超验叙事的渴求，最终在认知过载与逻辑死循环中，彻底失去了对现实的关照与改变现实的能力。

传统唯物主义的核心困境，在于未能打通“物质运动-生命演化-个体认知-社会历史”的完整逻辑链条：要么陷入机械的物质决定论，把人的意识、意义、价值当成了物质的被动附属品，无法解释个体的主观能动性与历史的创造性；要么陷入抽象的文化批判，脱离了唯物主义的本体论根基，最终滑向唯心主义的碎片化叙事。而作用唯物主义，正是为了填补这一理论缺口，回应普通人的现实与意义双重困境而诞生的。

## 二、双重批判：剥削结构的本质与唯心主义的欺骗机制
### （一）对剥削结构的系统性批判
我们以“自由力-剥削力”的核心框架，对人类社会的剥削结构做出全新的系统性解释：
1.  **剥削的本质**：剥削的核心是对人的自由力的压制——统治阶级通过对资源的垄断，让自身掌握的资源远超其自由力的可支配范围，而绝大多数被剥削者的资源被压缩至仅能维持生存，其自由发展的需求被彻底压制。
2.  **人类社会形态的演进规律**：从奴隶社会、封建社会到资本主义社会的发展，本质是**剥削力的逐步放松与自由力的持续扩张**。剥削力的收紧必然违背历史发展的总趋势，这也是以苏联为代表的集权式社会主义失败的核心原因——集权实质上造成了剥削力的反向收紧，压制了社会整体的自由力发展。
3.  **当代资本主义的隐蔽剥削**：当下的资本主义剥削，早已超越了单纯的剩余价值掠夺，形成了“物质剥削-意识形态控制”的双重闭环。它通过圈层化完成统治阶级的内部稳定，通过产业转移实现矛盾的跨国转嫁，更通过意识形态输出，让被剥削者陷入“梦游式生存”：把剥削阶级的精致生活标准当成自身的追求，把依附于特定剥削环境的意识形态当成自身的信念，最终成为剥削结构的自觉附庸，彻底失去了超越环境、改变现实的能力。
4.  **历史的真正动力**：历史从来不是“自在的螺旋式上升”，而是由广大人民的意识觉醒推动的——当沉睡的、被裹挟的个体，逐步苏醒并对“更合理的选择”做出筛选与行动时，历史才会真正向前发展。人民群众不是先天的“历史燃料”，而是“正在成为”的历史创造主体。

### （二）对唯心主义的再批判
唯心主义从来不是单纯的哲学认知偏差，而是与剥削结构深度合谋、维护统治秩序的意识形态工具，其核心欺骗机制分为两层：
1.  **转移注意力，消解现实关照**：通过“转生”“报应”“终极救赎”等超验命题，把人的注意力从现实的剥削关系、生存困境，转移到无法验证的超验领域。它让被剥削者把改变现实的希望，寄托于虚无的来世或超自然力量，最终彻底消解对现实剥削结构的批判意识与反抗能力。
2.  **制造认知过载，植入虚假答案**：唯心主义的核心命题，均是“绝对的无”“绝对的我”“万物的终极源头”这类涉及自指、无法在现实中验证的命题。这类命题既无法用形式逻辑完成无矛盾推导，也无法用根植于实践的辩证逻辑进行分析，只会让未经系统逻辑训练的人陷入无限的逻辑死循环与认知过载。而人对真理、确定性与意义的本能渴望，会让其迫切需要一个“终极答案”，此时唯心主义便会将维护剥削结构的叙事，包装成“终极真理”植入人的认知，最终完成对人的意识形态控制。

唯心主义的终极危害，在于它让人彻底脱离对现实的深切关照，要么陷入虚无主义的躺平，要么成为剥削结构的自觉附庸，这也是我们必须坚持唯物主义立场的核心现实原因。

## 三、作用唯物主义：核心体系、价值与开源迭代方向
### （一）核心理论体系
作用唯物主义以“作用”为核心一元范畴，构建了从宇宙本体论、生命演化论、个体认知论到社会历史观的完整贯通的哲学体系，核心框架如下：
1.  **本体论公理**
    - 物质与运动是一体的，作用是物质运动性的核心表现形式；
    - 物质、运动是第一性的，作用是第二性的；
    - 作用是不断生成的，是偶然与必然的统一，是偶然出现的“活力因子”的发展、扩张与生长。

2.  **作用生成的三阶段论（从无机界到意识诞生的完整演化链条）**
    - **第一阶段：交互的生成与作用的开始**：宇宙大爆炸前的奇点无交互即无作用，宇宙大爆炸后时空诞生、物质分化，不同物质间产生稳定交互，作用正式诞生。
    - **第二阶段：目标、效果的生成与自我维持**：无机界的交互是随机、趋于消亡的，而生命系统的本质，是在交互中生成了“自我维持的效果”，并由此诞生了“自我存续”的目标，形成了“效果-目标”的闭环，这是生命与无机界的本质边界。
    - **第三阶段：效果评价对目标的选择与意识的诞生**：意义的本质，是人在多目标决策中，受认知资源限制而形成的注意力机制，核心是对目标的效果评价与选择；当人脑将“持续发展”的属性内化到意义之中，意识便正式诞生。

3.  **意识发展的三阶段论**
    - **沉睡状态**：劳苦大众的核心意识状态，盲目、被动，完全被生存逻辑与剥削阶级意识形态裹挟，无自主的价值判断；
    - **梦游状态**：剥削阶级附庸的核心意识状态，以“自身存在的合法性”与“精致的物质生活”为核心诉求，完全依附于特定剥削环境，内核是剥削阶级的意识形态；
    - **觉醒（清醒）状态**：跨阶级的意识最高形态，自在、独立，以追求自由与发展的永恒为目标，具备对现实的深切关照与自主的价值判断能力。

4.  **完整哲学框架**
    - **世界观**：世界随着作用的生成、发展，呈现出从简单到复杂、从消极到积极、从必然走向消亡到趋向发展的永恒、从自在的存在迈向自在的自由的总趋势。
    - **方法论**：研究、认识事物，应关注其三重结构——**物质结构、作用结构与意义结构**；结构，是事物应对矛盾的作用机制。
    - **认识论**：人的认识发展，随着大脑的发育与矛盾的生成而不断进阶，完整链条为：认识特征→认识结构与交互→认识作用生成意义→意识诞生与作用物质化。
    - **历史观**：历史的核心是作用的历史，具备成果性与过程性两大属性；历史的本质是叙事性，是意识的历史，是人民群众意识逐步觉醒的过程。
    - **价值观**：世界是面向发展、迈向自由的；凡是有助于这一进程的事物，就是积极的、有价值的。

### （二）核心价值与时代意义
1.  **打通了自然与人类社会的一元逻辑链条**：以“作用”为核心范畴，贯通了从无机界物质运动、生命演化到人的意识、社会历史的所有层级，解决了传统唯物主义自然观与历史观割裂、物质运动与人的实践断层的核心难题。
2.  **为普通人提供了破解双重困境的认知工具**：既清晰揭示了剥削结构的本质，让普通人看清自身困境的结构性根源；也解释了意义的本质与意识觉醒的路径，让个体摆脱虚无主义与意识形态裹挟，找到改变现实、实现自身自由发展的清晰方向。
3.  **实现了马克思主义哲学的当代化拓展**：用系统科学、认知科学、演化生物学的最新成果，补充和拓展了经典马克思主义的核心框架，让其能够精准解释当代资本主义的新特征、新矛盾，重新激活了马克思主义的现实解释力。

### （三）待完善的逻辑开环与待验证假说
本体系仍处于迭代完善阶段，以下核心命题尚未完成完整的逻辑闭环与实证验证，也是本仓库开源迭代的核心方向：
1.  **意识的终极边界**：意识能否超越作用的存在、跳出作用的范畴实现绝对的自在自由，目前仅提出核心命题，未完成辩证论证与逻辑闭环；同时，“在作用的消极趋向下，人的意识必然存在消亡可能性”的判断，尚未完成严谨的辩证论证。
2.  **意义的起源假说**：智人意义机制的诞生，是同源起源还是多元起源，以及“智人与其他人种的种间斗争是意义诞生的核心动力”的假说，仍需古人类学、演化生物学的实证验证与逻辑补全。
3.  **意识范式的价值赋值方法**：如何建立可量化、可验证的意识范式价值赋值标准，仅提出了“跨民族双结构意识对比”的研究路径，未形成完整的体系化方法。
4.  **资本主义历史命运的完整论证**：仅提出“资本主义必然消亡”与“资产阶级在特定历史阶段存在必要性”的核心判断，未完成对资本主义消亡的具体历史进程、内在矛盾演化的完整论证。
5.  **人民主体性的现实落地路径**：仅锚定了“如何构建高效真实的人民意见表达体系”的核心问题，未完成从哲学理论到现实制度设计的完整落地路径。
6.  **国际竞争的意识形态本质**：仅提出“中美竞争的本质是隐蔽的意识形态之争”的判断，尚未完成完整的论证与现实分析。

### （四）核心发展方向
1.  **方向一：意识结构物质化与意识对作用的超越性**：完善意识发展的三阶段论，探索意识范式的价值赋值方法，打通个体意识觉醒到群体历史发展的逻辑链条，论证意识对作用的超越性的终极边界。
2.  **方向二：对剥削存在、唯心主义的批判与解构**：完善“自由力-剥削力”的分析框架，完成对当代资本主义新型剥削结构的系统性分析，结合认知科学成果完成对唯心主义的体系化再批判。
3.  **方向三：资本主义历史命运与当代国际竞争的意识形态本质**：完成对资本主义必然消亡的历史规律与资产阶级历史作用的完整论述，揭示中美竞争背后隐蔽的意识形态本质。

## 四、开源说明
本体系的核心初衷，是为普通人提供一套能够解释现实、改变现实的哲学认知工具，而非书斋里的抽象思辨。我们选择将其开源到GitHub社区，是因为哲学从来不是少数人的专利，它的生命力来自于现实的检验、大众的讨论与持续的迭代。

你可以通过以下方式参与本项目：
- 提出对理论体系的质疑、修正与补充建议；
- 补充相关的学术研究成果、实证案例与现实分析；
- 参与待完善命题的论证与拓展；
- 将本理论应用于具体的社会现实分析、文化研究、个人实践等场景。
